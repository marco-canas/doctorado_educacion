

## Módulo 1: Arquitectura del Texto Académico

*El objetivo es entender cómo se construye un artículo de investigación en educación.*

* **Gramática Funcional:** Reconocimiento de tiempos verbales clave (Presente Simple para hechos, Presente Perfecto para antecedentes y Pasado Simple para metodologías).
* **Conectores Lógicos (Transition Words):** Clasificación por función (adición, contraste, causa-efecto, conclusión).
* **Referentes Pronomiales:** Identificar a qué se refieren palabras como *it, they, which, whose* dentro de párrafos densos.
* **Sustantivación:** Cómo leer cadenas de sustantivos largas (típicas en títulos de papers).
